Sindicato Obras Sanitarias v0.0.1 Beta
------------------------------------

Instalación requerida
---------------------

1. Crear el directorio de trabajo y posicionarse en él.

2. Realizar una clonación del repositorio por medio de la terminal
    git clone https://github.com/REGaray/appAfiliadoSindicato.git

3. Instalar las siguientes dependencias:
    npm install -g electron
    npm init -y
    npm install --save-dev electron

4. Para ejecutar, utilizar el comando
    npm start
